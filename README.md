## 项目简介

本项目为html工具箱，用HTML、CSS和JavaScript写了一些小工具和小游戏，CSS和JavaScript都写在html文件内，一个.html文件即一个小工具或小项目，直接浏览器打开即可使用。

代码全部是我让ChatGPT GPT-4模型给我写的，我仅调整了代码格式和小部分注释。
